Troubleshooting

If the launch monitor will not go into ball detection mode:

  1. Try changing the club in GSPro
  2. If still not working, restart GSPconnect:
    • Go to Settings → System → Reset GSPro Connect
    • This app will automatically reconnect

About Infinite Tees

Infinite Tees is a golf simulator that supports the GSPro connector protocol.

Note: Only connect to one simulator at a time (either GSPro or Infinite Tees).

Device Settings

GSPro Settings

When enabled, automatically connects to GSPro when the device connects.

These settings rarely need to be changed. Default is localhost (127.0.0.1) on port 921.

Infinite Tees Settings

When enabled, automatically connects to Infinite Tees when the device connects.

Default is localhost (127.0.0.1) on port 999.

Camera Settings

When enabled, the connector will arm and update the external camera during ball-ready and shot events.

Example: `http://camera-box.local:8080`.
